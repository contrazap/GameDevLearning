# features/

Level 2 — one document per feature, generated on demand by [PLAN-GENERATOR.md](../PLAN-GENERATOR.md): a **guided plan**, **how-to plan**, or **user story** depending on skill rep counts (the fade). Naming: `<ID>-short-slug.md`, e.g. `A1.2-pipes-and-flow.md`.

Each document is also its feature's progress record: `**Next action:**` at the top, `## Session log` at the bottom, Claude keeps both current. Resume any morning by reading one file.

`S0-repos-and-spinup.md` is pre-written (it bootstraps everything else). Everything after it is generated when its turn on the [ROADMAP](../ROADMAP.md) arrives — never in advance.
